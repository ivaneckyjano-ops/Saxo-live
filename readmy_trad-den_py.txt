1. Importovanie potrebných knižníc:
	* `pandas` pre prácu s údajmi
	* `saxo_openapi` pre pripojenie k SAXO API
2. Nastavenie parametrov pre pripojenie k SAXO API:
	* `client_id`
	* `client_secret`
	* `access_token`
3. Pripojenie k SAXO API:
	* `saxo = saxo_openapi.SaxoOpenAPI(client_id, client_secret, access_token)`
4. Stiahnutie údajov:
	* `data = saxo.get_historical_prices("EURUSD", "D", "2022-01-01", "2022-12-31")`
5. Načítanie údajov do pandas dataframe:
	* `df = pd.DataFrame(data)`
6. Uloženie údajov do súboru CSV:
	* `df.to_csv("data.csv", index=False)`
7. Výpočet ukazovateľov:
	* `annualized_return`
	* `sharpe_ratio`
	* `percentualny_drawdown`
8. Vypísanie výsledkov:
	* `print(f'Annualizované zhodnotenie: {annualized_return_value:.2f}%')`
	* `print(f'Sharpe Ratio: {sharpe_ratio_value:.2f}')`
	* `print(f'Percentualný Drawdown: {percentualny_drawdown_value:.2f}%')`

Popis súboru `trad-den.py`:

Súbor `trad-den.py` je Python skript, ktorý sa používa na stiahnutie údajov z SAXO API a výpočet ukazovateľov pre analýzu finančných trhov. Skript sa pripojí k SAXO API, stiahne histórické ceny EURUSD a uloží ich do súboru CSV. Potom skript načíta údaje do pandas dataframe a vypočíta ukazovatele annualizovaného zhodnotenia, Sharpe ratio a percentuálneho drawdown. Nakoniec skript vypíše výsledky.

Štruktúra súboru `data.csv`:

Súbor `data.csv` je CSV súbor, ktorý obsahuje histórické ceny EURUSD. Súbor má nasledovnú štruktúru:

* `Date`: dátum
* `Open`: otvorená cena
* `High`: najvyššia cena
* `Low`: najnižšia cena
* `Close`: zatvorená cena

Používané knižnice:

* `pandas`: pre prácu s údajmi
* `saxo_openapi`: pre pripojenie k SAXO API

Verzia Pythonu:

* Python 3.x

Závislosti:

* `pandas`
* `saxo_openapi`

Inštalácia:

* `pip install pandas`
* `pip install saxo_openapi`

Použitie:
